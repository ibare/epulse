# 디자인 문서

## 디자인 결정 요약

ePulse는 "어두운 대시보드" 스타일의 금융 분석 도구 분위기를 기반으로, 다크 테마 위에 글래스모피즘(glassmorphism)과 미니멀한 데이터 시각화를 결합한 디자인을 채택했습니다.

핵심 디자인 원칙:
- **정보 밀도와 가독성의 균형**: 23개 변수와 22개 규칙을 한 화면에 표시하되, 시각적 과부하를 방지
- **인과관계의 직관적 전달**: 원인 -> 매개 -> 시장의 3열 흐름을 좌에서 우로 자연스럽게 읽히도록 배치
- **변화의 즉각적 인지**: 색상, 두께, 애니메이션을 통해 변화의 방향과 강도를 즉시 파악 가능

## 시각 구조

### 3열 레이아웃

```
+------------------+-------------------+-------------------+
|   좌측 패널      |    중앙 맵         |   우측 패널       |
|   (270px)        |    (flex-1)        |   (290px)        |
|                  |                    |                   |
|   슬라이더       |    원인 -> 매개    |   타임라인        |
|   프리셋 시나리오 |        -> 시장     |   예외 카드       |
|                  |                    |   노드 상세       |
+------------------+-------------------+-------------------+
```

이 구조를 선택한 이유:
- **좌측 입력, 우측 출력**: 자연스러운 정보 흐름(좌 -> 우)과 일치
- **중앙 맵 최대화**: 인과관계 그래프가 핵심 학습 요소이므로 가장 넓은 영역 할당
- **패널 고정 너비**: 맵 크기에 관계없이 조작/결과 패널의 일관된 사용성 보장

### React Flow 맵 배치

노드는 3개 열(x=0, x=350, x=700)에 배치됩니다:

- **원인 레이어 (x=0)**: 사용자가 직접 조작하는 근본 변수 (물가, 성장률, 유가 등)
- **매개 레이어 (x=350)**: 원인과 시장을 잇는 전달 변수 (금리, 환율 압력 등)
- **시장 레이어 (x=700)**: 최종 영향을 받는 시장 변수 (채권, 주식, 현물 등)

국가별로 수직 그룹핑(KR 상단, US 중간, EU 하단, GL 최하단)하여 지역 간 관계를 시각적으로 구분합니다.

## 색상 체계

### 배경 체계 (다크 테마)

```
--bg-primary:   #0f172a  (slate-900 계열, 메인 배경)
--bg-secondary: #1e293b  (slate-800 계열, 패널/카드 배경)
--bg-tertiary:  #334155  (slate-700 계열, 호버/액티브 상태)
```

어두운 배경을 선택한 이유:
- 금융 대시보드 컨벤션과 일치
- 밝은 데이터 포인트(노드, 엣지)의 대비 극대화
- 장시간 학습 시 눈의 피로 감소

### 지역 색상

| 지역 | 색상 | 용도 |
|------|------|------|
| KR (한국) | emerald (#10b981) | 노드 테두리, 엣지, 배지 |
| US (미국) | blue (#3b82f6) | 노드 테두리, 엣지, 배지 |
| EU | purple (#a855f7) | 노드 테두리, 엣지, 배지 |
| GL (글로벌) | amber (#f59e0b) | 노드 테두리, 엣지, 배지 |

지역 색상 선택 기준:
- **KR = emerald**: 브랜드 컬러이자 한국 시장의 주 색상으로, 앱 전체의 악센트 컬러
- **US = blue**: 달러/미국을 연상시키는 범용적 색상
- **EU = purple**: 한국/미국과 명확히 구별되는 색상
- **GL = amber**: 경고/주의를 연상시키는 색상으로, 글로벌 리스크 요소에 적합

### 상태 색상

| 상태 | 색상 | 용도 |
|------|------|------|
| 상승/긍정 | emerald (#10b981) | 양의 delta, 긍정적 방향 |
| 하락/부정 | rose (#f43f5e) | 음의 delta, 부정적 방향 |
| 중립 | slate (#64748b) | 변화 없음 |

## 노드 디자인

각 노드는 다음 요소로 구성됩니다:

- **지역 배지**: 좌상단에 KR/US/EU/GL 배지 (지역 색상)
- **라벨**: 변수명 (한국어)
- **상태 텍스트**: 현재 상태 (중립, 상승 압력 등)
- **강도 표시**: 점(dot) 기반 강도 표시기 (1~3개)
- **테두리 글로우**: 변화가 클수록 테두리에 글로우 효과 강화

입력 노드와 파생 노드의 시각적 구분:
- 입력 노드: 실선 테두리, 약간 밝은 배경 (사용자가 조절 가능함을 암시)
- 파생 노드: 점선 테두리, 표준 배경 (엔진이 계산한 값임을 암시)

## 엣지 디자인

인과관계 엣지의 시각적 속성:

- **두께**: 강도에 비례 (비활성 1px ~ 최대 3px)
- **색상**: 양의 관계는 emerald, 음의 관계는 rose
- **애니메이션**: 활성 엣지에 흐름(flow) 애니메이션 적용 (대시 오프셋 이동)
- **투명도**: 비활성 엣지는 낮은 투명도로 배경에 녹아들게 처리

이를 통해 "지금 활성화된 인과관계"를 한눈에 파악할 수 있습니다.

## 글래스모피즘

패널과 카드에 적용된 글래스 효과:

```css
background: rgba(15, 23, 42, 0.8);
backdrop-filter: blur(12px);
border: 1px solid rgba(51, 65, 85, 0.5);
border-radius: 16px;
```

글래스 효과를 선택한 이유:
- 레이어 간 깊이감 제공 (패널이 맵 위에 "떠 있는" 느낌)
- 배경 맵의 상태가 패널 뒤에서도 어렴풋이 보여, 전체 맥락 유지
- 모던한 대시보드 분위기 강화

## 반응형 대응

- **1024px 이상**: 3열 레이아웃 (좌측 패널 + 중앙 맵 + 우측 패널)
- **1024px 미만**: 좌측/우측 패널을 오버레이로 전환, 헤더에 토글 버튼 제공
- 핵심 학습 콘텐츠(맵)는 모든 화면 크기에서 항상 표시
