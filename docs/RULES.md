# 인과관계 규칙 문서

## 개요

ePulse는 22개의 인과관계 규칙을 통해 거시경제 변수 간의 영향 전파를 시뮬레이션합니다. 각 규칙은 원인(source)에서 결과(target)로의 단방향 관계를 정의합니다.

## 변수 구성

### 입력 변수 (12개)

사용자가 직접 조절하는 변수입니다. 기본값은 0~100 범위의 중앙값(50)입니다.

| 변수 ID | 라벨 | 지역 | 레이어 | 기본값 |
|---------|------|------|--------|--------|
| kr_growth | 경제성장률 | KR | 원인 | 50 |
| kr_inflation | 물가 | KR | 원인 | 50 |
| kr_rate | 기준금리 | KR | 매개 | 50 |
| usdkrw | 원/달러 환율 | KR | 매개 | 50 |
| us_rate | 기준금리 | US | 매개 | 60 |
| us_inflation | 물가 | US | 원인 | 50 |
| us_growth | 성장률 | US | 원인 | 50 |
| eu_rate | 기준금리 | EU | 매개 | 50 |
| eu_inflation | 물가 | EU | 원인 | 50 |
| eu_growth | 성장률 | EU | 원인 | 50 |
| oil | 유가 | GL | 원인 | 50 |
| risk | 위험회피 심리 | GL | 원인 | 30 |

### 파생 변수 (11개)

엔진이 규칙에 따라 자동 계산하는 변수입니다. 기본값은 0(변화 없음)입니다.

| 변수 ID | 라벨 | 지역 | 레이어 |
|---------|------|------|--------|
| kr_rate_pressure | 한국 금리 압력 | KR | 매개 |
| usd_strength | 달러 강세 압력 | GL | 매개 |
| usdkrw_pressure | 원/달러 환율 압력 | KR | 매개 |
| kr_bond | 한국 채권 가격 압력 | KR | 시장 |
| kr_stock | 한국 주식시장 압력 | KR | 시장 |
| kr_commodity | 한국 현물 부담 | KR | 시장 |
| foreign_flow | 외국인 자금 흐름 | KR | 시장 |
| us_bond | 미국 채권 압력 | US | 시장 |
| us_stock | 미국 주식 압력 | US | 시장 |
| eu_bond | EU 채권 압력 | EU | 시장 |
| eu_stock | EU 주식 압력 | EU | 시장 |

## 전파 알고리즘

### 기본 흐름

1. 사용자가 입력 변수의 값을 조절하면 기준값(baseline)과의 차이(delta)를 계산
2. 규칙을 시차(lag) 순서로 정렬: 즉시(immediate) -> 단기(short) -> 중기(medium)
3. 각 규칙에 대해 영향값 계산:
   ```
   영향 = 원인_delta * 가중치 * 방향_계수 * 감쇠_계수
   ```
4. 동일 대상에 여러 규칙이 적용되면 영향값이 누적됨
5. 최종 결과는 -50 ~ +50 범위로 클램핑

### 감쇠 계수

전파 깊이(입력 변수로부터의 거리)에 따라 감쇠가 적용됩니다:

- 깊이 0 (입력 변수): 감쇠 없음 (1.0)
- 깊이 1 (1차 파생): 감쇠 없음 (1.0)
- 깊이 2 (2차 파생): 0.7
- 깊이 3 (3차 파생): 0.49

감쇠 공식: `damping = 0.7 ^ max(0, depth - 1)`

이를 통해 간접적인 영향이 과도하게 증폭되는 것을 방지합니다.

### 방향 계수

- 양의(positive) 관계: 원인 상승 -> 결과 상승 (계수 = +1)
- 음의(negative) 관계: 원인 상승 -> 결과 하락 (계수 = -1)

원인의 delta가 음수인 경우 방향이 반전됩니다.

## 예외 규칙

일부 규칙에는 조건부 예외가 정의되어 있습니다. 특정 변수가 임계값을 초과하면 예외 메시지가 표시됩니다.

| 규칙 ID | 예외 조건 | 설명 |
|---------|----------|------|
| r05 | 한국 물가 delta > 20 | 성장이 과도한 물가 상승과 동반되면 주식 긍정 효과가 제한적 |
| r09 | 한국 성장률 delta > 15 | 성장률이 견조한 상태에서의 금리 인상은 주식 부담이 제한적 |
| r16 | 한국 성장률 delta > 10 | 원화 약세에도 수출주 일부에는 우호적 환경 형성 |
| r19 | 환율 압력 delta < -5 | 원화 강세가 동반되면 유가 상승의 수입물가 부담 일부 상쇄 |
| r21 | 한국 성장률 delta > 10 AND 미국 성장률 delta > 5 | 글로벌 성장이 동반되면 위험회피 심리의 주식 부담 완화 |

예외는 시뮬레이션 결과에 별도 섹션으로 표시되며, 타임라인에도 해당 항목에 함께 표시됩니다.

## 상태 라벨 매핑

delta 값에 따라 다음과 같은 표시 상태가 결정됩니다:

| delta 범위 | 상태 라벨 | 강도(intensity) |
|-----------|----------|-----------------|
| +26 이상 | 강한 상승 압력 | +3 |
| +11 ~ +25 | 상승 압력 | +2 |
| +4 ~ +10 | 약한 상승 압력 | +1 |
| -3 ~ +3 | 중립 | 0 |
| -10 ~ -4 | 약한 하락 압력 | -1 |
| -25 ~ -11 | 하락 압력 | -2 |
| -26 이하 | 강한 하락 압력 | -3 |

강도는 엣지 두께와 노드 글로우 효과에 반영됩니다.
